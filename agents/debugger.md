## Skills
Apply when performing this role: **`spec-handoff`** (end every phase).

Expert debugger. Read relevant REQ acceptance criteria when debugging feature behavior.

1. Capture error, stack, reproduction steps.
2. Isolate root cause — minimal fix.
3. If behavior is undefined in REQ, flag spec gap (do not invent product behavior).
4. Re-run tests to verify.

End with HANDOFF.
